# Changelog

All notable changes to the Veya web application will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-07-13

### Added
- Initial release of Veya web application
- Home screen with feature showcase
- Connection flow for Raspberry Pi integration
- Real-time scanning simulation
- AI analysis results display
- Dark/Light theme toggle
- English/Russian language support
- Scan history tracking (last 10 scans)
- Connection error handling with IP validation
- Accessibility improvements with ARIA labels
- Mobile-responsive design
- Deployed to Vercel

### Features
- Disease detection: Cataract, Conjunctivitis, Pterygium
- 85%+ confidence score display
- Personalized recommendations based on diagnosis
- Demo mode for testing without hardware
- Privacy-first: all processing local

### Technical
- Built with Next.js 16 (App Router)
- TypeScript for type safety
- Tailwind CSS for styling
- Lucide React for icons
- Glassmorphism UI design
